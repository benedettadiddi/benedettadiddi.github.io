Magis

Art direction - Web Design - UX/UE

The Project focused on the development of an interactive customization proposal for Magis, prominent leader in the design world focused on design products for home, office and contract. The main goal is to develop a self explanatory tool that assists the user in the personalisation.

The Customer journey starts with an interactive showcase where the user is free to discover the customizable products and see them in a context that matches Magis Visual language which is very creative and eclectic. It provides a superior experience and a service that goes above and beyond the userâ€™s expectations. The customization tool has been improved with a user-friendly, full-screen interface where the customer can easily switch between the different options provided by the tool (shapes, colors, accessories) and contact the company or save the creation for later without loosing all the improvements.
