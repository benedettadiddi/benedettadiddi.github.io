Interactive
Customization Tool

Magis is a prominent leader in the design world. The project focused on the development of a new interactive tool that assists the customer in the personalization of a product.

We develop an interactive showcase where the user is free to discover the customizable products in a specific enviroment that matches Magis Visual language.

The customization tool has been improved with a user-friendly, full-screen interface where the customer can easily switch between the different options provided by the tool (shapes, colors, accessories).
